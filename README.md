# futureshard_shield
This mod adds an amethyst powered kill field for destroying large clusters of mobs. This mod is compatible with VoxelLibre. I don't know about anything else, but it's written in lua...

The field is invisible, (not on purpose, sad) but because it simply punches hostile mobs within a certain radius millions of times, that mob will die in a shower of black hearts. Because it vaporizes arrows, it's probably good for rushing Pillager towers. (hint; it is). Crafting it requires Chorus fruit from the end, to ensure that no one can use it on the Ender Dragon. Crafting it is extremely difficult. It will steal one shard for every three seconds from your inventory, so better have some good amethyst growing blocks ready!

I tested it on (sorry snow golums) certain non-hostile mobs which I spawned in the village. And, (sorry villagers) it goes through walls. Have fun but don't touch golums. Lucky for me I was in creative.
